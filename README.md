## Scope

This repository contains only the unified portal, static search index, service boundaries and shared navigation entry points. Product article bodies remain in their existing repositories so established URLs and search indexing are preserved.

ChongGrok is not affiliated with OpenAI, xAI, X, Anthropic or Google. This project does not provide API credits, SMS verification or bulk registration. ChatGPT, Grok and Claude do not offer ready-made accounts; Gemini is the only product with a separate ready-made annual account option.
