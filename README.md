# ChongGrok AI Membership Guides

Unified GitHub Pages directory for the ChongGrok ChatGPT and SuperGrok membership knowledge bases.

- Website: <https://he20000405-pixel.github.io/>
- ChatGPT guide: <https://he20000405-pixel.github.io/chatgpt-plus-pro-china-guide/>
- Grok guide: <https://he20000405-pixel.github.io/supergrok-china-guide/>
- Main service: <https://chonggrok.com/>

## Scope

This repository contains only the unified portal, static search index, service boundaries and shared navigation entry points. Product article bodies remain in their existing repositories so established URLs and search indexing are preserved.

ChongGrok is not affiliated with OpenAI, xAI, X, Anthropic or Google. This project does not provide API credits, ready-made accounts, SMS verification or bulk registration.

## Local build

```powershell
bundle install
bundle exec jekyll build
bundle exec jekyll serve
```

See [PUBLISHING.md](PUBLISHING.md) for deployment and indexing steps.
